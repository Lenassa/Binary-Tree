Constructeur d'un arbre pour un individu à partir d'une forêt d'ancêtres.

Paramètres
indL'individu racine
ancLa forêt d'ancêtres
Précondition
ind est présent dans anc
Postcondition
L'arbre a ind pour racine et contient tous les descendants de ind présents dans anc

Complexité -temporelle : O(nAnc*nbIndMaxDegre) avec nAnc le nombre d'individus dans anc nbIndMaxDegre le plus grand nombre d'ind par degré dans anc -spatiale : O(nAnc + 2*nbIndMaxDegre + O(ajouter))
